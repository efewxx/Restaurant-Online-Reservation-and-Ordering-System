# Restaurant Online Reservation and Ordering System
<img width="1168" height="784" alt="xmxX3" src="https://github.com/user-attachments/assets/ed9d8067-e114-4998-9d85-47782c0932a9" />


A modern web-based application that allows customers to reserve tables and order food online, with a powerful admin panel for restaurant management.

##  Features

### Customer Side
- User registration and login system
- Browse restaurant menu with categories
- Table reservation with date and time selection
- View, update, and cancel reservations
- Add food items to cart
- Order summary with total price calculation
- Responsive and user-friendly interface

### Admin Side
- Manage menu categories
- Add, update, and delete menu items
- View and manage all reservations
- View customer orders
- User and reservation management

## 🛠 Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP / Node.js (you can specify)
- **Database**: MySQL
- **Authentication**: Session or JWT based
- **Validation**: JavaScript validation on forms

## 🚀 How to Run

1. Clone the repository
2. Import the database (`restaurant_db.sql`)
3. Configure database connection
4. Start the local server
5. Open in your browser
